# Hotel Management System

A desktop-based Hotel Management System developed using Python, Tkinter, and MySQL.  
This application helps manage hotel operations such as customer records, room booking, billing, and room allocation through an interactive graphical user interface.

---

## Features

- User Login Authentication
- Customer Management System
  - Add Customer
  - Update Customer Details
  - Delete Customer Records
  - Search Customer Information
- Room Booking Management
  - Check-in / Check-out
  - Room Allocation
  - Meal Selection
  - Billing Calculation
- MySQL Database Integration
- Interactive GUI using Tkinter

---

## Technologies Used

- Python
- Tkinter
- ttk Widgets
- MySQL
- SQL
- DBMS

---

## Project Modules

### Login System
Secure login interface for accessing the application.

### Customer Management
Stores and manages customer details with CRUD operations.

### Room Booking
Handles room reservations, room type selection, and booking records.

### Billing System
Calculates room charges, taxes, and total booking cost.

---

## Database

The project uses MySQL Server to store:

- Customer Records
- Booking Information
- Room Details
- Billing Data

## How to Run the Project

### 1. Install Python

Download and install Python on your system.

### 2. Install Required Libraries

```bash
pip install mysql-connector-python
```

### 3. Create the MySQL Database

Create the required database and tables in MySQL Server.

### 4. Run the Project

```bash
python hotel.py
```
